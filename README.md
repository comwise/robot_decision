# robot_decision
robot decision, this is a frame, you can receive new task and state, then can dispatcher to nav and action. if robot state changed, then handle any state and error state

# depend
- ubuntu/windows
- cmake >= 3.2.0
- c++ >= 14

# how to build
1. mkdir build
2. cd build 
3. cmake ..
4. make 

# frame 
- commuication
- state 
- handle task
- handle fault
- handle nav/action
- ...
